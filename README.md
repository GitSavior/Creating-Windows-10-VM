# Creating-Windows-10-VM

Show how to download a Windows 10 ISO and install a windows 10 VM with that ISO.

## Steps

### Step 1: Downloading the Windows 10 ISO

#### Google Search and Select "Download Windows 10 Disc Image (ISO File)" Link

![image](https://github.com/GitSavior/Creating-Windows-10-VM/assets/162067776/7995abb5-3e4f-454b-a94c-e9289bd0d236)
![image](https://github.com/GitSavior/Creating-Windows-10-VM/assets/162067776/11ef8ca4-240a-4416-9b52-f03f51ad5f4a)

#### Click "Download Now" button to download the Windows 10 installation media

![image](https://github.com/GitSavior/Creating-Windows-10-VM/assets/162067776/d60cc327-0cee-4f9d-a9fc-8c327b5878ad)

#### Open the file "MediaCreationTool22H2.exe"

![image](https://github.com/GitSavior/Creating-Windows-10-VM/assets/162067776/62faad25-7759-4198-9888-63dc611d1405)

#### Click "Accept" to agree to the License Terms

![image](https://github.com/GitSavior/Creating-Windows-10-VM/assets/162067776/f2adac27-be8a-44f3-996c-1f43b8da356b)

#### Select the "Create installion media (USB flash drive, DVD, or ISO file) for another PC" Option and then click "Next"

![image](https://github.com/GitSavior/Creating-Windows-10-VM/assets/162067776/d3419753-8a57-43d3-a9ea-8c82666e08eb)

#### Uncheck the "Use the recommnded options for this PC" if need to select the options in down boxes Language, Edition, and Architure the Windows preference to be used and the click "Next"

![image](https://github.com/GitSavior/Creating-Windows-10-VM/assets/162067776/82f20a55-82ea-4710-951e-8a8fa9242ee7)

##### Select "ISO file" and then click "Next"

![image](https://github.com/GitSavior/Creating-Windows-10-VM/assets/162067776/5bebc48c-f6d9-4a6d-8a71-1e89c760a7f8)

#### A window will pop-up to Select a path to download the windows.iso too. 

![image](https://github.com/GitSavior/Creating-Windows-10-VM/assets/162067776/d0c6cac0-860e-458c-a791-eb5a12eda539)

#### Click "Finish"

![image](https://github.com/GitSavior/Creating-Windows-10-VM/assets/162067776/390f86bc-328d-43b6-92d1-395c895258bd)

#### The Windows.iso

![image](https://github.com/GitSavior/Creating-Windows-10-VM/assets/162067776/e57b1ca7-11ab-4324-bec2-00b6b7f99e98)

### Step 2: Setting up the Windows 10 virtual machine

#### Open Virtual box and select "New"

![image](https://github.com/GitSavior/Creating-Windows-10-VM/assets/162067776/e0376680-52ad-4ea3-b249-c7429e945356)

#### Create a name and select the ISO image "Windows.iso" downloaded in the first step and then click "Next"

![image](https://github.com/GitSavior/Creating-Windows-10-VM/assets/162067776/e034fb77-2d09-48df-866f-aa59c35617dc)

#### To enable copy and paste between the Host (your computer) and Guest (VM), check the box labeled "Guest Additions". Change your username, password, and any other preferences. Click "Next"

![image](https://github.com/GitSavior/Creating-Windows-10-VM/assets/162067776/26f99d47-c4c9-4943-9a2b-0c2b533c4c3d)

#### Choose how much preferred hardware to allocate for the VM. Then click "Next"

![image](https://github.com/GitSavior/Creating-Windows-10-VM/assets/162067776/f4845110-aa94-42e5-9356-f280ca88d618)

#### Choose how much hard disk space to reserve for the VM. Then, click "Next".

![image](https://github.com/GitSavior/Creating-Windows-10-VM/assets/162067776/e1e9f983-0c1a-4eca-b7fb-9504853d3498)

#### Click "Finish"

![image](https://github.com/GitSavior/Creating-Windows-10-VM/assets/162067776/6f34b456-2e7d-4650-987d-08bdbee26b16)

#### The virtual computer will launch, and the setup will begin.

![image](https://github.com/GitSavior/Creating-Windows-10-VM/assets/162067776/9e3622cf-deef-4611-a12c-4f5450a7e787)

#### The setup should complete without any more interaction. 

![image](https://github.com/GitSavior/Creating-Windows-10-VM/assets/162067776/df484274-539d-4b38-9df6-50518bb42377)
